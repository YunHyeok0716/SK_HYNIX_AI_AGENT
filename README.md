# AI_AGENT
AI Agent 사전 과제_FIT_ME_UP

- 사용자가 원하는 패션 스타일(미니멀, 스트릿, 캐주얼 등)을 요구하면, 사용자의 옷장 및 신발장을 기반으로 최적의 옷, 신발, 액세서리를 조합하여 매일 아침에 맞춤형 OOTD를 추천해주는 AI Agent입니다.

- 사용자가 자신의 옷장, 신발장, 수납장을 카메라로 스캔하면, 스마트 비전 기술을 통해 현재 사용자가 가지고 있는 패션 아이템의 종류, 색상, 재질을 자동으로 데이터화하여 등록합니다.

- 사용자가 "오늘 출근용 미니멀 룩 추천해 줘", "주말 데이트용 캐주얼 스타일 조합해 줘"라고 요구하면, 보유한 아이템들을 기반으로 상·하의, 신발, 액세서리까지 완벽하게 조합한 룩을 웹 대시보드에 제안합니다.

- 추가적으로 스마트한 옷장 관리 기능을 제공합니다. 사용자가 보유 중인 아이템의 활용도를 분석하여, "현재 옷장에 미니멀 스타일 아이템이 70%를 차지하고 있습니다. 가지고 계신 슬랙스와 매치하기 좋은 로퍼를 추가하면 매력도가 높아집니다"와 같이 쇼핑 아이템을 추천해주는 기능을 제공합니다.
